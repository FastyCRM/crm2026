# CORE — Ядро CRM2026

## Назначение
Папка `/core` содержит **общее ядро системы**, используемое всеми частями проекта.

CORE отвечает за:
- подключение БД
- сессии
- авторизацию
- CSRF
- базовые утилиты
- bootstrap проекта

---

## Основные принципы
- `declare(strict_types=1);`
- функциональный подход
- никаких HTML-шаблонов
- никаких зависимостей от модулей
- ядро не знает UI

---

## Типовые функции
- `db()` — подключение к БД (PDO)
- `auth_is_logged_in()`
- `auth_user_role()`
- `auth_user_theme()`
- `csrf_token()`
- `clean()`
- `redirect()`

---

## Структура (логическая)

/core/
│
├── bootstrap.php # Точка инициализации
├── db.php # Подключение к БД
├── auth.php # Авторизация и роли
├── csrf.php # CSRF-защита
├── helpers.php # Утилиты
└── README.md


(физическая структура может отличаться, логика фиксирована)

---

## Ограничения
CORE НЕ должен:
- выводить HTML
- содержать layout
- знать о модулях
- зависеть от admin/site
