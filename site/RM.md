# SITE — Публичная часть CRM2026

## Назначение
Папка `/site` содержит **публичную часть проекта**:
- лендинги
- мини-магазины
- статические страницы
- публичные формы

---

## Принципы
- данные получаются только из CORE
- работа преимущественно через JSON
- минимум динамики
- без прямой логики БД

---

## Архитектура
- site НЕ знает о модулях админки
- site НЕ содержит административной логики
- изменения данных инициируются через CORE

---

## Ограничения
SITE НЕ должен:
- содержать админский UI
- напрямую работать с БД
- использовать модули из `/modules`
